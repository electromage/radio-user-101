# Radio Communications 101

A quick-start guide to two-way radio etiquette and basic procedures.

---

## Core Principles

- **Think before you key up.** Know what you're going to say before you press the transmit button.
- **Be brief, be clear, be done.** Radio time is shared — keep transmissions short.
- **Listen first.** Before transmitting, listen for a few seconds to make sure the frequency is clear. Never transmit over someone else.

---

## How to Use the Radio

1. **Press and hold** the Push-to-Talk (PTT) button before speaking.
2. **Pause a beat** after pressing PTT before you start talking — the first fraction of a second can get clipped.
3. **Speak clearly** at a normal pace across the face of the microphone (not directly into it).
4. **Release PTT** when you're done speaking.
5. **Hold the radio straight up and down** (vertical). Tilting it reduces antenna efficiency and weakens your signal.
6. **Keep the radio away from large metal objects** — vehicles, metal fences, machinery, etc. Metal near the antenna reflects and absorbs your signal, cutting your range.

---

## Calling Another Station

Use this format:

> **[Who you're calling], this is [who you are]. Over.**

Examples:

- "Base, this is Unit 4. Over."
- "Alpha team, this is Bravo team. Over."

The called station responds:

> "Unit 4, this is Base. Go ahead."

If no one responds, wait 10–15 seconds and try again.

## Acknowledging and Responding

| You want to...            | Say                          |
| ------------------------- | ---------------------------- |
| Confirm you received      | "Copy" or "Roger"            |
| Say yes                   | "Affirmative"                |
| Say no                    | "Negative"                   |
| Ask them to repeat        | "Say again"                  |
| Signal you're done talking for the moment | "Over"        |
| Signal the conversation is finished       | "Out"         |
| Acknowledge an instruction you'll carry out | "Wilco" (will comply) |

**Never say "Over and out."** "Over" means you expect a reply; "Out" means you don't. Pick one.

---

## Giving a Brief Report (SALUTE Format)

When reporting what you observe, the SALUTE format keeps it organized:

| Letter | Stands for | Example                        |
| ------ | ---------- | ------------------------------ |
| S      | Size       | "Three vehicles"               |
| A      | Activity   | "Moving east on Highway 9"     |
| L      | Location   | "Grid reference 123 456"       |
| U      | Unit/ID    | "Marked with blue logos"       |
| T      | Time       | "Observed at 1430"             |
| E      | Equipment  | "Two sedans, one pickup truck" |

Put it together on the radio:

> "Base, this is Unit 4. Spot report, over."
> *(wait for acknowledgment)*
> "Three vehicles, moving east on Highway 9, grid 123 456, marked with blue logos, observed at 1430, two sedans and one pickup. Over."

---

## Giving Your Position

Use whatever reference system your group has agreed on (grid coordinates, landmarks, mile markers). Always confirm the format so there's no confusion.

> "Unit 4 is at the junction of Main Street and 5th Avenue."

---

## On-Air Etiquette

- **Don't use names.** Use callsigns or unit designators.
- **Don't chitchat.** Save casual conversation for face-to-face.
- **Don't swear or use profanity.** On many frequencies it's illegal; on all of them it's unprofessional.
- **Don't interrupt.** If someone is mid-transmission, wait for "Over" before you key up. If it's urgent, say "Break, break" to signal priority traffic.
- **Identify yourself** every time you transmit. Don't assume people recognize your voice.
- **Use plain language.** Avoid jargon or slang that others might not know. Say "I understand" instead of obscure codes.
- **Emergency traffic takes priority.** If you hear "Mayday" or "Emergency," clear the channel immediately.

---

## Emergency Calls

Prefix with the urgency level:

| Prefix               | Meaning                             |
| -------------------- | ----------------------------------- |
| **"Mayday, Mayday, Mayday"** | Life-threatening emergency |
| **"Pan-Pan, Pan-Pan, Pan-Pan"** | Urgent but not immediately life-threatening |
| **"Break, break"**   | Priority message, clear the channel |

Format:

> "Mayday, Mayday, Mayday. This is Unit 4. We have a medical emergency at grid 123 456. One person injured, requesting immediate assistance. Over."

---

## Radio Check

To test whether your radio is working:

> "Any station, this is Unit 4. Radio check, over."

Respond with signal quality:

| Response              | Meaning                  |
| --------------------- | ------------------------ |
| "Loud and clear"      | Strong signal, clear audio |
| "Readable but weak"   | Can understand, low signal |
| "Unreadable"          | Cannot understand        |
